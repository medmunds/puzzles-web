Sokoban

Push each barrel (brown circle) into a target (grey circle). Barrels can be pushed up, down, left or right, but not into another barrel or the wall.

Use the arrow keys or numeric keypad to move. Alternatively, click on the grid to move toward where you clicked.

Status: This is an experimental, unfinished puzzle. It may have bugs or unexpected behavior. Random game generation is too simplistic to be credible, but the rest of the gameplay works well enough to use it with hand-written level descriptions.
