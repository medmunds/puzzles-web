Puzzles

from Simon Tatham’s portable puzzle collection

This is a collection of some forty logic puzzles and games, all completely free to play, right here in your web browser. (No ads. No trackers. For real!)

Each puzzle offers multiple difficulty levels and variations and unlimited, randomly generated games. You’ll never run out of puzzles to solve.

Some may be familiar: Sudoku fans will enjoy Solo. Sweeping for Mines is a classic way to waste pass the time. And puzzles like Fifteen and Pegs have been played (in wood and stone form) for literally centuries.

Other puzzles are clever ideas from around the globe, new twists on old standards, and maybe even your next favorite just waiting to be discovered. (Have you tried ?)

This web app should run in any modern browser. You can use a touch screen, mouse or keyboard. It even works offline (look for “install app” or “add to home screen”). See the source code about box for credits and licenses.

Enjoy!
