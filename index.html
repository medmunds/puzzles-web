Puzzles

from Simon Tatham’s portable puzzle collection

This is a collection of some forty logic puzzles and games. There are familiar classics similar to Sudoku (“Solo”), Mastermind (“Guess”) and Minesweeper (“Mines”), plus some clever challenges that may be completely new to you.

Each puzzle offers multiple difficulty levels and variations. With unlimited, randomly generated games, you’ll never run out of puzzles to solve.

This web app should run in any modern browser. It supports touch screens, mouse and keyboard control. It even works offline (you can “install app” or “add to home screen”).

And it’s completely free: no in-app purchases, no ads, no trackers. (For real! All thanks to open source contributors: details in the source code about box.)

Enjoy!
